# 信号源

用于无线电测向运动中，被找寻的盒子。

主要使用STM32单片机和CDCE913频率发生器。并且使用了两个9013三极管和一个2SD882功率管以及自己缠绕的线圈组成的功放电路。

具体的细节请看博客：https://blog.csdn.net/x_fengmo/article/details/143115911

我们燕山大学“燕之波”无线电测向协会基于哈工大制作信号源的方法，改用STM32单片机制作信号源
